Política de Privacidad

1. Introducción

AdShield es una extensión de navegador diseñada para bloquear anuncios y mejorar la experiencia del usuario en la web. Respetamos tu privacidad y nos comprometemos a no recopilar, almacenar ni compartir ningún dato personal.

2. Datos que NO recopilamos

Esta extensión NO recopila ni almacena información personal, historial de navegación, direcciones IP u otra información identificable del usuario.

3. Permisos requeridos

Para funcionar correctamente, AdShield solicita los siguientes permisos:

  • webRequest y webRequestBlocking: Para bloquear anuncios antes de que se carguen en las páginas web.
  • Permiso de host (<all_urls>): Para analizar y bloquear anuncios en cualquier página web visitada.

Estos permisos se utilizan exclusivamente para el propósito de bloqueo de anuncios y no se usan para recopilar datos personales.

4. Seguridad

AdShield no transmite ni comparte datos con terceros. No se conecta a servidores externos ni carga scripts remotos.

5. Cambios en esta política

Podemos actualizar esta política ocasionalmente. Se recomienda revisar esta página periódicamente para estar informado de cualquier cambio.
